# バッジ画像制作仕様書

## 概要

ワイン記録アプリ「Wine Mapper」で使用するバッジ・ランクアイコンの画像制作依頼です。
ユーザーのコレクション欲を刺激する、美しく統一感のあるバッジセットを制作してください。

---

## 全体デザインガイドライン

### スタイル
- **テイスト**: 高級感のあるフラットデザイン。ワインラベル風の上品さ。写実性重視。
- **アプリのカラーパレット**:
  - メイン背景: `#faf6f0`（クリーム）
  - テキスト: `#1c1c18`（ほぼ黒）
  - アクセント1: `#561922`（ワインレッド）
  - アクセント2: `#c9a84c`（ゴールド）
  - サブ: `#722f37`（バーガンディ）
  - ボーダー: `#d8c1c2`（ローズグレー）
- **トーン**: 落ち着きがありつつも、達成感・コレクション欲を感じるデザイン
- **禁止事項**: チープなクリップアート風、過度にポップな色使い

### 画像仕様
- **フォーマット**: PNG（透過背景）
- **サイズ**: 512×512px（アプリ内で縮小表示するため高解像度で納品）
- **形状**: 円形またはメダル型（全バッジで統一）
- **ファイル命名規則**: `{カテゴリ}-tier{数字}.png`
  - 例: `volume-tier1.png`, `rank-tier3.png`

### ティア共通の視覚ルール
各カテゴリ内でTier 1→7に向かって以下の変化を段階的に表現してください:
- **色の深さ**: 薄い・淡い → 深い・リッチ
- **装飾の豊かさ**: シンプル → 紋章・オーナメント追加
- **素材感**: マット → メタリック / 光沢
- **サイズ感（モチーフ）**: 小さく控えめ → 大きく堂々
- **文字**: 一切使用しない。デザインだけでわかるようなものにすること。

## 参考: アプリのスクリーンショット

アプリの全体的なトーンは、クリーム背景にワインレッドのアクセント、ゴールドの装飾を使った落ち着いたデザインです。バッジはこのトーンに調和しつつ、獲得した時の「嬉しさ」を感じるデザインにしてください。
