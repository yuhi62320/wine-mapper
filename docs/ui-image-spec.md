# UI装飾画像 制作仕様書

## 概要

ワイン記録アプリ「Wine Mapper」の各画面で使用するUI装飾画像の制作依頼です。
現状テキストやアイコンのみで寂しい箇所に、アプリの世界観を補強するビジュアルを追加します。

---

## 全体デザインガイドライン

### スタイル
- **テイスト**: 高級感のあるフラットデザイン。ワインラベル風の上品さ。写実性重視。
- **アプリのカラーパレット**:
  - メイン背景: `#faf6f0`（クリーム）
  - テキスト: `#1c1c18`（ほぼ黒）
  - アクセント1: `#561922`（ワインレッド）
  - アクセント2: `#c9a84c`（ゴールド）
  - サブ: `#722f37`（バーガンディ）
  - ボーダー: `#d8c1c2`（ローズグレー）
- **トーン**: 落ち着きがありつつも、ワインの世界への没入感を感じるデザイン
- **文字**: 一切使用しない。デザインだけで伝わるものにすること。
- **禁止事項**: チープなクリップアート風、過度にポップな色使い

### 画像仕様
- **フォーマット**: PNG（透過背景）
- **解像度**: 各画像の指定サイズの2倍（Retina対応）で納品
  - 例: 指定300×400 → 納品600×800

## 制作リスト（15枚）

### 1. ホームページ — ヒーロー画像

| ファイル名 | `hero-home.png` |
|---|---|
| サイズ | 1200×400 |
| 用途 | ホーム画面最上部の背景画像 |
| 内容 | ブドウ畑のパノラマ風景。朝霧がかかった早朝のイメージ。クリーム＋ゴールド系のトーン。下部にグラデーション（→`#faf6f0`に溶け込む）を付けて、その上にテキストが載る想定 |

### 2. ログインページ — 背景画像

| ファイル名 | `bg-login.png` |
|---|---|
| サイズ | 800×1200 |
| 用途 | ログイン画面の全体背景 |
| 内容 | ワインセラーの奥行きを感じる構図。暗めでムーディー。ボトルが棚に並ぶシルエット。上にログインフォーム（白系カード）が重なるため、全体的に暗く落ち着いたトーン |

### 3. ホームページ — ステータスカード画像（3枚）

ホーム画面中央の「ワイン数 / 探索国数 / 品種数」カードの装飾画像。
カード背景やアイコン部分に配置する小さなイラスト。

| ファイル名 | `stat-wines.png` |
|---|---|
| サイズ | 200×200 |
| 用途 | 「ワイン数」カードの装飾 |
| 内容 | ワインボトルが3〜4本並ぶミニイラスト。コレクション感。ワインレッド〜バーガンディの色調 |

| ファイル名 | `stat-countries.png` |
|---|---|
| サイズ | 200×200 |
| 用途 | 「探索国数」カードの装飾 |
| 内容 | 小さな地球儀、またはパスポートにスタンプが押されたイメージ。ゴールド〜ブラウン系 |

| ファイル名 | `stat-grapes.png` |
|---|---|
| サイズ | 200×200 |
| 用途 | 「品種数」カードの装飾 |
| 内容 | 色とりどりのブドウの粒が集まるイラスト。紫・緑・ルビー色が混在。自然な彩り |

### 4. ワインタイプ別プレースホルダー（7枚）

ワインの写真が未登録の場合に表示するデフォルト画像。
ワインカード・ワイン詳細ヘッダーの両方で使用。

| ファイル名 | サイズ | タイプ | 内容 |
|---|---|---|---|
| `placeholder-red.png` | 300×400 | 赤ワイン | 赤ワインボトルのエレガントなイラスト。深いルビー色。落ち着いた雰囲気 |
| `placeholder-white.png` | 300×400 | 白ワイン | 白ワインボトル。明るいゴールド〜ペールイエロー。爽やかなトーン |
| `placeholder-rose.png` | 300×400 | ロゼ | ロゼワインボトル。淡いサーモンピンク。フェミニンで軽やか |
| `placeholder-sparkling.png` | 300×400 | スパークリング | シャンパンボトル。泡が立ち上るエフェクト。華やかさ |
| `placeholder-fortified.png` | 300×400 | フォーティファイド | ポートワイン風の太いボトル。琥珀〜ダークブラウン。重厚感 |
| `placeholder-dessert.png` | 300×400 | デザート | 375ml小瓶のデザートワイン。蜂蜜色〜黄金色。甘美なイメージ |
| `placeholder-orange.png` | 300×400 | オレンジ | オレンジワインボトル。アンバー〜オレンジ色。自然派・クラフト感 |

### 5. 品種図鑑 — ヘッダー画像

| ファイル名 | `header-grapes.png` |
|---|---|
| サイズ | 1200×300 |
| 用途 | 品種図鑑ページの上部ヘッダー背景 |
| 内容 | 多品種のブドウが横に並ぶ水彩画風イラスト。左から緑系（白品種）→紫系（赤品種）へのグラデーション。自然で美しいボタニカルアート風 |

### 6. 記録ページ — ヘッダー画像

| ファイル名 | `header-new-wine.png` |
|---|---|
| サイズ | 800×200 |
| 用途 | ワイン記録ページの上部装飾 |
| 内容 | ワインを注ぐ瞬間のイラスト。ゴールド系の色調。期待感・特別な体験の始まりを感じるトーン |

### 7. 空のコレクション — 案内画像

| ファイル名 | `empty-collection.png` |
|---|---|
| サイズ | 400×400 |
| 用途 | ワインが0本の時にホームやコレクション一覧で表示 |
| 内容 | 空のワインラック。1本だけスペースが光っているイメージ。「最初の1本を記録しよう」という気持ちを誘うデザイン。寂しさではなく、これから始まるワクワク感 |
