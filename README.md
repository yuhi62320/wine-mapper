# Wine Mapper 🍷🗺️

ワインのラベルを撮影して情報を記録し、世界地図上で飲んだワインの産地をコレクションするアプリ。

## コンセプト

- **スキャン & ログ**: ワインラベルを撮影 → AI が情報を自動抽出 → ワンタップで記録
- **アロマ記録**: UC Davis アロマホイールベースの香り表現システム（日本語対応）
- **ワインマップ**: Fog of War 方式で産地を探索。飲むほど地図が解放される
- **ゲーミフィケーション**: XP・ランク・バッジ・ストリークでコレクション欲を刺激
