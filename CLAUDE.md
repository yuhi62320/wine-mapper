# Wine Mapper - Project Instructions

## Project Overview
ワインの写真を撮ってラベル情報を読み取り、飲んだワインを記録・コレクションするアプリ。
地図上で産地を可視化し、ゲーミフィケーションでコレクション欲を刺激する。

## Language
- ドキュメント: 日本語
- コード・コメント: 英語
- コミットメッセージ: 英語

## Key Design Principles
- コレクション欲の最大化（ゲーミフィケーション重視）
- 香りの表現にこだわる（アロマホイール＋日本語対応）
- 最小限の操作でワインをログできるUX（10秒以内の基本ログ）
- Fog of War方式のマップで探索欲を刺激
